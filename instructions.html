Instructions

Dear participant , please read the instructions before the experiment.

Thank you for completing the pre-experiment questionnaire.Now,please go through the instructions below for the rest of the experiment interface.

Through this experiment,We would be evaluating the comparative performance of two different Copy-Paste Techniques - the traditional Ctrl C+V technique and a new technique called AutocomPaste .

In the experiment you would be using both these techniques in different scenarios of copy-pasting either within the same windows or between different windows.

To give you an idea of how the interface looks,please find the screenshot below :

Picture of interface

You would be using four different scenarios for this experiment.Please see the video below to understand more about each one of them:

AUTOCOMPASTE - Between documents

TRADITIONAL - Between documents

We hope the above instructions are clear with respect to the Techniques and the functioning of the interface. However , we would request you to please keep the following points in mind while conducting the experiment :

1.Please read the technique every time that you would be using for conducting the trial before proceeding with the trial.

2.You would be given three trials each for every scenario to copy-paste with both the Traditional Ctrl C/Ctrl V technique and with the new AutocomPaste technique.

3.The Granularity of the text (Phrase/sentence/Paragraph) would change every time you undertake a subsequent trial within each technique.

4.We request you to please turn your Hand phones to silent mode and refrain from interacting with other participants or other people during the experiment. We request you to please not talk any calls or messages during the experiment.

Thank you and Best of Luck!
